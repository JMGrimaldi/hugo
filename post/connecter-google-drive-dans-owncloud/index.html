Connecter Google Drive dans Owncloud

17 novembre 2013

Google ayant compliqué la procédure d’accès à ses API, voici comment récupérer un ID/secret pour connecter Google Drive dans Owncloud 5 et plus :

  1. Allez sur la console Google Cloud ;
  2. Créez un nouveau projet, choisissez un nom (p.ex. API Project) et un ID pour ce projet ;
  3. Entrez dans la console du projet en cliquant sur son nom, puis APIs & auth > Registered apps, et choisissez Register app  ;
  4. Choisissez un nom pour cette application (p.ex. Owncloud), comme plate-forme Web Application, et cliquez sur Register  ;
  5. Sur l’écran suivant qui vous demande de choisir le mode d’autorisation, sélectionnez OAuth 2.0 Client ID  ;
  6. Renseignez dans REDIRECT URI l’adresse de la page d’admin Owncloud (http:///index.php/settings/personal) puis cliquez sur Generate et notez les valeurs de Client ID et Client Secret  ;
  7. Dans l’admin Owncloud, dans la rubrique Stockage externe ajoutez une source Google Drive, et dans Client ID/secret saisissez les valeurs notées à l’étape précédente.
