Pixma MG5250 sur Ubuntu

10 août 2015
 Partager sur TwitterTweeter  Partager sur Facebook  

Cette page explique comment installer l’imprimante-scanner Canon Pixma MG5250 sur Ubuntu (testé sur 10.04, 10.10, 13.04).

Récupération des pilotes Canon

  • Télécharger MG5200series-printer_driver.tar et MG5200series-scanner_driver.tar depuis www.canon.fr : Linux IJ Printer Driver (3.40) et Linux IJ Scanner Driver (1.60)
  • Aller dans le répertoire Téléchargements/

Installation de l’imprimante

Prérequis : le package libtiff4 doit être installé.

% mkdir printer ; cd printer
% tar xvf ../MG5200series-printer_driver.tar
% tar xvzf cnijfilter-mg5200series-3.40-1-deb.tar.gz
% cd cnijfilter-mg5200series-3.40-1-deb/
% sudo ./install.sh

Suivre les instructions.

Vérifier que l’imprimante apparaît bien dans Administration > Impression

Si vous rencontrez un problème la doc est dans le fichier guidemg5200series- pd-3.40-1_en.tar.gz (à décompresser avec tar xvzf <fichier>)

NB : le driver Canon limite la résolution à 600dpi

% cd ../.. ; rm -rf printer

Installation du scanner

% mkdir scanner ; cd scanner
% tar xvf MG5200series-scanner_driver.tar
% tar xvzf scangearmp-mg5200series-1.60-1-deb.tar.gz
% cd scangearmp-mg5200series-1.60-1-deb/
% sudo ./install.sh
  • Lancer Gimp (si besoin : apt-get install gimp), si dans Fichier > Créer, ‘ScanGear MP’ n’apparaît pas sous ‘Capture d’écran’ :
    • Quitter Gimp
    • sudo ln -sf /usr/bin/scangearmp /usr/lib/gimp/2.0/plug-ins
    • Relancer Gimp
  • Fichier > Créer > ScanGear MP ; un message d’erreur apparaît mais l’autodétection du scanner doit vous montrer votre Canon, sélectionnez-le et la fenêtre de numérisation apparaît.
