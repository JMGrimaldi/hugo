JMparse

9 août 2002
 Partager sur Twitter  Partager sur Facebook  
Cet article est obsolète. Le site utilise désormais le générateur Hugo.

JMparse c’est le script grâce auquel toutes les pages de ce site ont été générées ; en particulier le menu sur la gauche.

Il permet de générer des pages partageant la même présentation, à partir d’une part de fichiers de contenus et d’autre part d’un modèle, qui est une page HTML contenant des balises spéciales. Par exemple, $content$ sera remplacé lors de la génération par le fichier de contenu, $menu$ par un menu, etc.

C’est un petit script bien utile, que vous pouvez télécharger ci-dessous.

> télécharger JMparse 1.1 (18 ko)

Anciennes versions

> télécharger JMparse 1.0 (19 ko)
