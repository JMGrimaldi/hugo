Installer WordPress avec SQLite

20 mai 2013
 Partager sur Twitter  Partager sur Facebook  

Cet article explique comment installer un blog WordPress utilisant une base SQLite (fichier plat) au lieu d’une base MySQL. Cela permet d’avoir tout le blog dans un seul répertoire et facilite les sauvegardes. Il faut bien sûr que l’hébergeur supporte SQLite.

Quelle que soit la version de WordPress que l’on souhaite utiliser à la fin, il faut commencer par installer la version 3.1.2.

  1. Téléchargez WordPress 3.1.2 et le plugin PDO (SQLite) 2.7.0 ;

  2. Décompressez les 2 fichiers. Vous devriez maintenant avoir deux répertoires wordpress et pdo-for-wordpress ;

  3. Placez le répertoire pdo et le fichier db.php, situés dans le répertoire pdo-for-wordpress, dans le répertoire wordpress/wp-content ;

  4. Créez un répertoire database sous wordpress/wp-content ;

  5. Renommez le fichier wp-config-sample.php situé dans le répertoire wordpress en wp-config.php, puis ouvrez ce fichier dans un éditeur de texte ;

  6. Ajoutez la ligne suivante 

    define('DB_TYPE', 'sqlite'); // utiliser sqlite
    après les lignes 
    /** The Database Collate type. Don't change this if in doubt. */
define('DB_COLLATE', '');
    pour donner
    /** The Database Collate type. Don't change this if in doubt. */
define('DB_COLLATE', '');
define('DB_TYPE', 'sqlite'); // use sqlite

  7. Remplacez les lignes 

    define('AUTH_KEY', 'put your unique phrase here');
define('SECURE_AUTH_KEY', 'put your unique phrase here');
define('LOGGED_IN_KEY', 'put your unique phrase here');
define('NONCE_KEY', 'put your unique phrase here');
define('AUTH_SALT', 'put your unique phrase here');
define('SECURE_AUTH_SALT', 'put your unique phrase here');
define('LOGGED_IN_SALT', 'put your unique phrase here');
define('NONCE_SALT', 'put your unique phrase here');
    avec vos propres clefs aléatoires générées sur en allant sur https://api.wordpress.org/secret-key/1.1/salt/, et enregistrez le fichier ;

  8. Ouvrez le fichier wordpress/wp-content/pdo/wp_install.php dans un éditeur de texte et remplacez la ligne 

    $message = __('<strong><em>Note that password</em></strong> carefully! It is a <em>random</em> password that was generated just for you.');
    par la ligne 
    $message = __('<strong><em>Note that password</em></strong> carefully! It is a <em>random</em> password that was generated just for you.<br><br>   ' . $random_password);
    et enregistrez le fichier ;

  9. Si vous utilisez un hébergement distant, transférez-y tout le répertoire wordpress ;

  10. Donnez les droits d’écriture au serveur web sur le répertoire wordpress/wp-content/database. Dans le doute, modifiez les permissions en 777 ;

  11. Dans votre navigateur web, ouvrez l’adresse http://yourdomain/wp-admin/install.php pour démarrer l’installation WordPress ;

  12. Choisissez un username et un password puis cliquez sur le bouton install. Si la page affiche une flopée de code PHP ignorez cela et descendez tout en bas ;

  13. Notez votre password puis cliquez sur le lien login situé en bas de la page ;

  14. Entrez vos username et password, vous devriez arriver sur le panneau de configuration de WordPress ; de là vous pourrez mettre à jour vers la dernière version en Français.
