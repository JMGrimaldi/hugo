Formation POV #1: Intro

28 avril 2000
 Partager sur TwitterTweeter  Partager sur Facebook  

Présentation

PovRay (Persistence Of Vision Raytracer) est un programme d’image de synthèse basé sur un langage de description.
Ainsi, on écrit un fichier .pov décrivant une scène (objet, éclairage et appareil-photo), puis on donne le nom du fichier en argument à Pov, qui crée en retour l’image correspondante.

Par exemple, on peut écrire un fichier essai.pov contenant le texte suivant :

sphere {0,1 pigment {green 1}}
light_source {<-5,8,-9> rgb 1}
camera {location -3*z look_at 0}

Ci-dessus le résultat de la commande povray +i essai.pov (le nom de l’exécutable pouvant varier suivant les systèmes).

Installation

La dernière version est la 3.1g (était en fait, car la 3.7 est sortie depuis). Elle est notamment disponible sous linux et windows. À noter que cette dernière version dispose d’un excellent éditeur.

Installation windows

  • télécharger l’archive ici ;
  • lancer povwin3.exe et suivre les instructions ;
  • dans l’éditeur, pour lancer un rendu, cliquer sur “Run” ou faire ALT+G.

Installation linux

  • télécharger l’archive ici ;
  • la décompresser : tar xvzf povlinux.tgz ;
  • aller dans le répertoire ainsi créé : cd povray31/ ;
  • exécuter le script d’install : ./install ;
  • pour lancer un rendu, sous X, taper x-povray +d +i nomfichier.pov
    (on peut modifier les options par défaut dans ~/.povrayrc).
    Note 1 : La version incluse dans les distributions debian (de la potato à la sid) est la 3.02, depuis laquelle des changements importants ont eu lieu. Préférez donc une installation à partir du tar.gz.
    Note 2 : Si /usr/local/man/man1/ n’existe pas, créez-le avant d’exécuter ./install : mkdir /usr/local/man/man1.
