Définir la vue par défaut des dossiers Windows

11 août 2015

Comment faire, dans l’explorateur de fichiers Windows 7, pour que la vue du dossier actuel (tri par Nom ou par Type, largeur des colonnes, etc.) s’applique à tous les dossiers ?

  • Ajustez la vue souhaitée ;
  • Appuyez sur la touche Alt pour faire apparaître le menu ;
  • Allez dans Outils > Options des dossiers…, onglet Affichage ;
  • Appuyez sur le bouton Appliquer aux dossiers et confirmez.
