Booter le Samsung 370R5E-S02 sur un support USB

2 octobre 2013
 Partager sur Twitter  Partager sur Facebook  

Le PC Samsung 370R5E-S02 est livré avec Windows 8 préinstallé, et les réglages par défaut du BIOS ne permettent pas de booter sur un support USB.

Pour changer cela :

  1. Bootez le PC et appuyez sur F2 dès le splash screen jusqu’à l’apparition de la config du BIOS ;
  2. Dans l’onglet Advanced, mettez Fast BIOS Mode à [Disabled] ;
  3. Dans l’onglet Boot, mettez Secure Boot à [Disabled] ;
  4. Appuyez sur F10 pour sauvegarder la configuration et redémarrer ;
  5. À nouveau, appuyez sur F2 pour entrer dans la config du BIOS ;
  6. Dans l’onglet Boot, sélectionnez Boot Device Priority et dans Boot Option #1 choisissez votre support USB (NB : avec un disque externe, il peut falloir un reboot supplémentaire pour qu’il soit reconnu), puis appuyez sur F10.
